# Entwicklungsumgebung

Ihnen wird von der Schule eine Entwicklungsumgebung in Form von Docker-Containern bereitgestellt.

Diese Entwicklungsumgebung stellt Ihnen folgende Komponenten zur Verfügung:

- Einen webgestützten Vs-Code Editor mit Plugins.

- Ein Proxyserver, der dafür sorgt, dass die Umgebung auf einem Webserver einer URL erreichbar ist.

Es steht Ihnen frei, diese oder adäquate andere Komponenten direkt auf Ihrer Hardware zu verwenden. Eine direkte Verwendung hätte den Vorteil, dass Sie mit beliebigen Editoren direkt auf Ihren Code zugreifen können. Allerdings würde dies einen sehr großen Aufwand bedeuten, welcher aber auch viel wertvolle Erfahrung generieren wird.
